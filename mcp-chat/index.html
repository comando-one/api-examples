🤖 Chat IA via MCP

CLI · roda no terminal (Node 18+)

Converse em português e o Gemini chama as ferramentas do @comando.one/mcp-server, que executa ações reais na API do Comando.One.

geminimcpfunction calling ~50 toolszero deps

Como rodar

# chaves (mesmas envs do projeto)
export GEMINI_API_KEY="..."            # aistudio.google.com/apikey
export COMANDO_API_KEY="cmd_live_..."  # Configurações → API Keys

node mcp-chat/chat.mjs

Ou copie .env.example para .env nesta pasta e preencha — o script carrega sozinho.

Quer no navegador?

Esta versão é CLI (sobe o MCP como subprocesso stdio e lê env — coisas de Node). Para conversar direto no navegador, use o 💬 Chatbot IA (navegador), que usa o MCP remoto (api.comando.one/mcp) via streaming. Veja o código no README.
